How to start Selenium in Python
1. Install Selenium
pip install selenium
2. Install WebDriver Manager
pip install webdriver-manager

Note: You can also install packages from PyCharm File Menu > Settings > Python Interpreter then click on + button to install packages (just like installing packages from Eclipse Marketplace)
